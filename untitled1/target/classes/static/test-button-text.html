🔍 问题分析

可能的原因：

1. CSS样式问题：按钮基础样式设置了 text-transform: uppercase

2. 字体大小问题：表格中按钮字体过小

3. 颜色对比问题：文字颜色与背景色对比度不够

4. 文字间距问题：letter-spacing 设置过大

✅ 解决方案

修复CSS样式：

• 为表格中的按钮添加 text-transform: none

• 调整 letter-spacing 为正常值

• 确保文字颜色对比度足够

• 优化按钮大小和间距
